1.4.0 - 2020.12.20
------------------
KSP 1.11 Compatibility
All SoundingRocket parts are now more buoyant.
Sounding rocket nose cones are now packed with foam, to make them super floaty.
Sounding rocket parts are now a lot more durable.
To encourage more sounding rocket use around Kerbin, the following changes have been implemented:
- Sounding rocket experiments are now repeatable.
- Sounding rocket experiments are now restricted to atmosphere only.
- Sounding rocket experiments now give significantly more science.
All rocket motors can toggle their auto-drop so that you can use the last motor as ablative landing gear.
Adjusted the 0.625m nose cone's parachute dynamics to be more in line with a Mark 1 stock chute

0.5.6 - 2017.01.01
-------------------
Some minor cost adjustments (a few things are now less expensive)
There are two new parachutes - a cargo drogue chute and a large cargo chute.  Use these to avoid chute spam and to land truly massive payloads.
Grid fins will now properly disable control surfaces when not deployed.  Note that since the module has changed, existing action groups will not work - you will need to reassign them.
The 0.625m SRB has a fixed surface attach node
